# Facebook Hacker Cup 2017

:trophy: Solutions to problems from the Facebook Hacker Cup 2017 :trophy:

## Usage

- If is a C++ file (.cpp):
```bash
g++ file.cpp -o main
./main
```

- If is a C file (.c):
```bash
gcc file.c -o main
./main
```

- If is a Java file (.java):
```bash
javac file.java
java file
```
